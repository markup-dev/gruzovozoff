# Скриншоты для README.md

Эта папка предназначена для хранения скриншотов проекта "Грузовозофф".

## Необходимые скриншоты для README:

1. **main-page.png** - Главная страница с формой входа
2. **register.png** - Страница регистрации пользователей
3. **create-application.png** - Форма создания заявки на перевозку
4. **user-applications.png** - Список заявок пользователя
5. **admin-panel.png** - Панель администратора с карточками
6. **filtering.png** - Фильтрация и поиск в админ панели
7. **mobile-view.png** - Мобильная версия с бургер меню

Каждый скриншот должен быть:
- Формат: PNG
- Разрешение: минимум 1200px по ширине
- Качество: высокое
- Фон: с данными для демонстрации функциональности

Скриншоты должны показывать реальную работу системы с тестовыми данными.
